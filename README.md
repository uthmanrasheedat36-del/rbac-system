# RBAC System (Role-Based Access Control)

## 📌 Description
This project implements a simple Role-Based Access Control (RBAC) system using Python. It assigns roles to users and grants permissions based on those roles.

## 👥 Roles
- Admin: create, read, update, delete
- Editor: read, update
- Viewer: read
